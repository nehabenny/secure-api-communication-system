OWASP A02: Cryptographic Failures Mitigation

Key Exchange (ECDH): Derives a shared secret over an insecure channel without transmitting the secret itself.
Data Integrity (HMAC): Ensures the payload hasn't been tampered with. The server verifies the signature using the shared secret.
Replay Protection: Timestamps (≤ 30s) are included in the HMAC payload to reject captured & replayed requests.
HTTPS / TLS: All API communication is encrypted in transit using TLS with a self-signed certificate.
